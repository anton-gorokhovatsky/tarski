# Tarski static site

Статичный сайт для GitHub Pages.

## Обновление v5

- Мобильное floating-меню осталось стеклянным, но кнопка-гамбургер теперь использует форму точки из логотипа Tarski (`assets/menu-shape.svg`).
- Активный пункт в раскрытом мобильном меню автоматически прокручивается в видимую область при скролле страницы.
- Если открыть меню уже в середине страницы, горизонтальная плашка сразу центрируется на текущем разделе.

## Публикация на GitHub Pages

Замени в репозитории файлы:

```text
index.html
styles.css
script.js
assets/menu-shape.svg
```

Остальные ассеты можно не трогать, если они уже актуальны.

После замены сделай `Commit changes`. GitHub Pages обновится автоматически.
