# Tarski static site

Статичный сайт для GitHub Pages.

## Обновление v6

- Базовое мобильное меню в шапке снова отображается в столбик.
- После прокрутки inline-меню появляется закреплённая горизонтальная glass-плашка без hamburger-кнопки.
- Закреплённое мобильное меню автоматически прокручивается к активному разделу при скролле контента.
- Индикатор активного пункта теперь использует SVG-форму точки из буквы `i` логотипа Tarski: `assets/menu-shape.svg`.
- Такой же SVG-индикатор используется в desktop-меню.

## Публикация на GitHub Pages

Замени в репозитории файлы:

```text
index.html
styles.css
script.js
assets/menu-shape.svg
```

Остальные ассеты можно не трогать, если они уже актуальны.

После замены сделай `Commit changes`. GitHub Pages обновится автоматически.
