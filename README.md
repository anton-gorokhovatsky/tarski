# Tarski static site

Статичный сайт для GitHub Pages.

## Обновление на GitHub

Замените в корне репозитория файлы:

- `index.html`
- `styles.css`
- `script.js`
- папку `assets/` при необходимости

После этого сделайте commit. GitHub Pages обновит сайт автоматически.

## Что есть в версии v3

- Desktop-меню справа закреплено при скролле.
- Индикатор активного пункта меню обновляется при скролле.
- На mobile основное меню остаётся в шапке строкой.
- Когда строка меню уезжает выше экрана, появляется закреплённый hamburger.
- При клике hamburger раскрывается в чёрное горизонтальное меню по мотиву Figma-макета `24:146`.
- Пункты mobile-меню закрывают меню после перехода к секции.

## Структура

```text
index.html
styles.css
script.js
assets/
  logo.svg
  footer-logo.svg
  artist-cover.png
  favicon.svg
```


## Обновление v4

В мобильном плавающем меню чёрная плашка заменена на стеклянную: полупрозрачный фон, blur/saturate, светлая обводка и мягкая тень. Для браузеров без `backdrop-filter` есть плотный светлый fallback.
